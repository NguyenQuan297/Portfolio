## Tech Stack

- React + Vite
- TailwindCSS
- Framer Motion
- AOS (scroll-triggered animations)

## Setup

```bash
npm install
npm run dev
```

The app is configured to use:
- `public/Portfolio_image.jpg` as the HERO background
- `public/hero-avatar.png` as the HERO avatar
